# Specmatic Reports

Generated Specmatic evidence is written here by `scripts/run-specmatic-mode.sh`
or `scripts/run-specmatic-mode.ps1`.

Expected generated files:

- `contract-test-report.html`
- `positive-only-report.html`
- `resiliency-report.html`
- `contract-test-output.txt`
- `positive-only-output.txt`
- `resiliency-output.txt`

Screenshots captured from Specmatic Studio, generated HTML reports, or GitHub
Actions logs should be stored under `screenshots/`.
